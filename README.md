# Sand Scape Explorer

Explore stunning sand landscapes and desert vistas. Discover the world's most breathtaking sandscapes through an interactive digital museum featuring sand samples from around the globe.

## Tech Stack

- **React 18** — UI framework
- **TypeScript 5** — Type safety
- **Vite 5** — Build tool & dev server
- **Tailwind CSS 3** — Utility-first styling
- **shadcn/ui** — Accessible component library
- **React Router 6** — Client-side routing
- **TanStack Query** — Async state management
- **Zod** — Schema validation

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```
